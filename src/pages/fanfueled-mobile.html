FanFueled Mobile

FanFueled doesnt have native apps for mobile devices, so we had to implement a webapp that is mobile compatible.
FanFueled events and Engage are two apps under FanFueled apps that are integrated with seamless navigation. First step for the mobile solution was to create a navigation that easy to use on mobile and is seamless as on original webpages.

Mobile Navigation

This is a javascript implementation to simulate mobile navigation similar to native mobile apps.

FanFueled Events Mobile

Events pages were not mobile friendly, As the mobile traffic increased for events pages, we needed to develop a mobile solution thats doesnt take toll on mobile data and also provides necessary information. So with above goals in mind we had to redo event pages to be mobile ready. First steps were to reduce information overload and provide necessary details along with easy to use ticket widget. This ticket widget helped users to choose tickets with ease and checkout easily. A new user friendly mobile checkout page was also created.

FanFueled Engage Mobile

Engage is big and complex app that needs a proper mobile app for its proper use, which is in works. But we wanted to create an intermediate solution thats not complex but is easy to implement and solves the mobile problems.
A responsive web solution was implemented with CSS and JavaScript. Also the web page partials are created to be served on demand.
